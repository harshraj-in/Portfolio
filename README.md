# 🌌 Personal Portfolio — Harsh Raj

Welcome to the repository of my personal portfolio website. This project is a high-fidelity, interactive 3D web experience built to showcase the intersection of computer science, creative graphic design, and modern web engineering.

🔗 **Live Link:** harshraj-in.netlify.app

---

## 🎨 Technology Stack & Features

*   **3D Graphics & Engine:** [Three.js](https://threejs.org/) rendering a real-time crystal geometry field with interactive refractive glass elements (`MeshPhysicalMaterial`) and velocity-driven particles.
*   **Kinetic Motion & Animation:** [GSAP (GreenSock)](https://greensock.com/) synced with GSAP `ScrollTrigger` for scroll-linked layout fades and staggered animations.
*   **Buttery Scrolling:** [Lenis Smooth Scroll](https://github.com/darkroomengineering/lenis) integration for trackpad/scroll wheel physics normalization.
*   **Aesthetics:** Curated modernist glassmorphic styling, custom differences-blend mouse cursor trail, and deep space color scheme.
*   **Architecture:** Clean codebase with fully decoupled, organized files ([index.html](index.html) and [style.css](style.css)).
